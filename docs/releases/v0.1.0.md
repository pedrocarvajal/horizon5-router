## Summary

Initial release of Horizon Connect, a Django-based API router with WebSocket support. This release establishes the foundational architecture with Docker containerization, Celery task queue, Redis integration, and comprehensive REST API capabilities designed for scalable real-time communication.

## Breaking Changes

None (Initial Release)

## New Features

- **Django REST API Framework**: Full REST API support with Django 5.1 and Django REST Framework 3.15.0
- **WebSocket Support**: Real-time bidirectional communication via Django Channels 4.1.0, Channels-Redis, and Daphne ASGI server
- **Docker Containerization**: Production-ready Docker configuration with docker-compose.yml for orchestration
- **Celery Task Queue**: Asynchronous task processing with Celery 5.4.0 integration
- **Redis Integration**: Redis 5.0.0 support for caching, message broker, and channel layer
- **Database Support**: PostgreSQL database support via psycopg2-binary
- **API Documentation**: Automatic OpenAPI/Swagger documentation generation with DRF Spectacular
- **CORS Configuration**: Cross-origin resource sharing support for frontend integration
- **Development Tools**: Django Extensions for enhanced development experience
- **Environment Management**: Secure configuration via python-dotenv
- **Data Validation**: Pydantic 2.12.3 for robust data validation and serialization
- **Core App Structure**: Modular app structure with routing, consumers, serializers, pagination, and permissions
- **Project Commands**: Custom management commands infrastructure
